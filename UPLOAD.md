# Zo upload je naar Vercel (belangrijk!)

Je screenshot toont de **oude** pagina. Vercel heeft nog **oude** bestanden.

## Wat je moet doen

1. Open GitHub repo **status-utrecht** (of jouw repo)
2. Klik **Add file** → **Upload files**
3. Sleep de **hele map** `utrecht-status` van je pc erin  
   (of minimaal: `index.html`, `js/status.js`, `css/status.css`, `api/`, `server/`)
4. **Commit changes**
5. Vercel → wacht op **Ready** (automatisch)
6. Open je site met **Ctrl+F5**

## Goed geüpload als je dit ziet

- Groene tekst: **"UI build 5"** onder de titel
- FiveM-kaart met groot **5 / 128**, groene balk, **ONLINE** linksboven
- **Geen** paarse tekst "5 / 128 spelers" meer
- **Geen** server-icoon links met "Online" rechts

## Nog steeds oud?

- Verkeerde repo gekoppeld in Vercel
- Alleen 1 bestand geüpload i.p.v. hele map
- Browser cache: incognito venster proberen
